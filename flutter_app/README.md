# ReddMobile Flutter App

This directory contains the Flutter client for ReddMobile.

## Quick start

```bash
flutter pub get
flutter run
```

## Validation

```bash
flutter analyze
flutter test
```

## Notes

- Chain data is sourced from `https://blockbook.reddcoin.com` via `lib/services/blockbook_service.dart`.
- Architecture and contributor onboarding live in the repository root (`ARCHITECTURE.md`, `CONTRIBUTING.md`).
